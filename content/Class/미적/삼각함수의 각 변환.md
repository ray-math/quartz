>[!Tip] 각 변환 공식 
>1. 각을 $\frac{\pi}{2} \times n \pm \theta$ 꼴로 바꾼다. (단, $n$은 정수이고 $\theta$는 예각이라 가정한다.)
>2. $\frac{\pi}{2} \times n \pm \theta$이 몇 사분면의 각이냐에 따라 삼각함수를 변환한다.
>3. $n$이 짝수이면 삼각함수를 변환하지 않고, $n$이 홀수이면 삼각함수를 변환($\sin \to \cos$, $\cos \to \sin$, $\tan \to \cot$)한다.
>4. $\theta$의 부호와 관계없이 반드시 양의 부호로 변환한다.

## 예제

2. $\cos\left( \frac{\pi}{2} + \theta \right)$ [^2]  
3. $\tan\left( \frac{\pi}{2} + \theta \right)$ [^3]

8. $\sin(\pi - \theta)$ [^8]  
9. $\cos(\pi + \theta)$ [^9]  
10. $\cos(\pi - \theta)$ [^10]  
11. $\tan(\pi + \theta)$ [^11]  
12. $\tan(\pi - \theta)$ [^12]  
13. $\sin\left( \frac{3\pi}{2} + \theta \right)$ [^13]  
14. $\sin\left( \frac{3\pi}{2} - \theta \right)$ [^14]  
15. $\cos\left( \frac{3\pi}{2} + \theta \right)$ [^15]  
16. $\cos\left( \frac{3\pi}{2} - \theta \right)$ [^16]  
17. $\tan\left( \frac{3\pi}{2} + \theta \right)$ [^17]  
18. $\tan\left( \frac{3\pi}{2} - \theta \right)$ [^18]  
19. $\sin(2\pi + \theta)$ [^19]  
20. $\sin(2\pi - \theta)$ [^20]  
21. $\cos(2\pi + \theta)$ [^21]  
22. $\cos(2\pi - \theta)$ [^22]  
23. $\tan(2\pi + \theta)$ [^23]  
24. $\tan(2\pi - \theta)$ [^24]  
25. $\sin\left( -\frac{\pi}{2} + \theta \right)$ [^25]  
26. $\sin\left( -\frac{\pi}{2} - \theta \right)$ [^26]  
27. $\cos\left( -\frac{\pi}{2} + \theta \right)$ [^27]  
28. $\cos\left( -\frac{\pi}{2} - \theta \right)$ [^28]  
29. $\tan\left( -\frac{\pi}{2} + \theta \right)$ [^29]  
30. $\tan\left( -\frac{\pi}{2} - \theta \right)$ [^30]  

[^1]: $\sin\left( \frac{\pi}{2} + \theta \right) = \cos \theta$  
[^2]: $\cos\left( \frac{\pi}{2} + \theta \right) = -\sin \theta$  
[^3]: $\tan\left( \frac{\pi}{2} + \theta \right) = -\cot \theta$  
[^4]: $\sin\left( \frac{\pi}{2} - \theta \right) = \cos \theta$  
[^5]: $\cos\left( \frac{\pi}{2} - \theta \right) = \sin \theta$  
[^6]: $\tan\left( \frac{\pi}{2} - \theta \right) = \cot \theta$  
[^7]: $\sin(\pi + \theta) = -\sin \theta$  
[^8]: $\sin(\pi - \theta) = \sin \theta$  
[^9]: $\cos(\pi + \theta) = -\cos \theta$  
[^10]: $\cos(\pi - \theta) = -\cos \theta$  
[^11]: $\tan(\pi + \theta) = \tan \theta$  
[^12]: $\tan(\pi - \theta) = -\tan \theta$  
[^13]: $\sin\left( \frac{3\pi}{2} + \theta \right) = -\cos \theta$  
[^14]: $\sin\left( \frac{3\pi}{2} - \theta \right) = -\cos \theta$  
[^15]: $\cos\left( \frac{3\pi}{2} + \theta \right) = -\sin \theta$  
[^16]: $\cos\left( \frac{3\pi}{2} - \theta \right) = \sin \theta$  
[^17]: $\tan\left( \frac{3\pi}{2} + \theta \right) = \cot \theta$  
[^18]: $\tan\left( \frac{3\pi}{2} - \theta \right) = -\cot \theta$  
[^19]: $\sin(2\pi + \theta) = \sin \theta$  
[^20]: $\sin(2\pi - \theta) = -\sin \theta$  
[^21]: $\cos(2\pi + \theta) = \cos \theta$  
[^22]: $\cos(2\pi - \theta) = \cos \theta$  
[^23]: $\tan(2\pi + \theta) = \tan \theta$  
[^24]: $\tan(2\pi - \theta) = -\tan \theta$  
[^25]: $\sin\left( -\frac{\pi}{2} + \theta \right) = -\cos \theta$  
[^26]: $\sin\left( -\frac{\pi}{2} - \theta \right) = -\cos \theta$  
[^27]: $\cos\left( -\frac{\pi}{2} + \theta \right) = \sin \theta$  
[^28]: $\cos\left( -\frac{\pi}{2} - \theta \right) = -\sin \theta$  
[^29]: $\tan\left( -\frac{\pi}{2} + \theta \right) = -\cot \theta$  
[^30]: $\tan\left( -\frac{\pi}{2} - \theta \right) = \cot \theta$
